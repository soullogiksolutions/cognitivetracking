Available Cognitive Assessments

Backward Digit Span

Backward Digit Span is a classic working memory test where participants recall digits in reverse order. The script adapts difficulty for personalized brain training while ensuring all data stays private and offline. Typical completion takes 8-10 minutes.

Forward Digit Span

Forward Digit Span assesses short-term memory by having participants recall sequences as given. Our privacy-first offline tool ensures that no personal data is tracked or stored. Typical duration is 8-10 minutes.

Auditory N-Back

Auditory N-Back challenges working memory with customizable auditory sequences presented offline. We guarantee zero data retention, fully respecting user privacy. Typical test length is 10-15 minutes.

Reading Span

The Reading Span task combines sentence judgment and letter memorization to track working memory capacity with full data privacy assured by offline operation. Duration is approximately 15-20 minutes.
